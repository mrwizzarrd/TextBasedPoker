# TextBasedPoker

## What this is
Right now this project is dedicated to learning applications of object oriented programming and class based code design

### Things Learned from this Project (So far):
- Workflow Management
- Basic Data Structures/Algorithms
- Refresher on Object Oriented Design and OOP Concepts such as Inheritence/Polymorphism
- Clean User Interface Design

## How to use

### Windows
1. Ensure Java (JDK 21 or newer recommended) is installed.
2. Open Command Prompt and type the following commands:
```bat
git clone https://github.com/mrwizzarrd/TextBasedPoker.git
cd TextBasedPoker 
build.bat 
```

### Linux/macOS
1. Ensure Java (JDK 21 or newer recommended) is installed.
2. Open Terminal and run the following commands
```bash 
git clone https://github.com/mrwizzarrd/TextBasedPoker.git
cd TextBasedPoker
make
make run
```
